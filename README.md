# Macao Romana
Proiect cu principii OOP in C# Visual Studio.

Implementari:
- umfla cu 2/3
- schimba simbolul
- stai o tura
- restartare joc la pierdere/castig

Versiunea asta NU are:
- buton de Striga Macao cand jucatorul mai are doar o carte in mana
- posibilitatea de a juca pachetele (mai multe carti de acelasi numar dar simbol diferit)
- posibilitatea de a pune AS cand celalalt jucator a pus AS
- carte de start pe tabla cand incepe jocul
- mai multe ferestre (start screen, player edit screen)
- carte de stopare a umflarii
- interfata grafica personalizata (folosesc strict Toolbox-ul si imaginile cartilor in resursele proiectului)

Pachetul nu contine jokeri.
