# Macao Romana
Proiect cu principii OOP in C# Visual Studio.

Implementari:
- umfla cu 2/3
- schimba simbolul
- stai o tura
- restartare joc la pierdere/castig
- buton de Striga Macao cand jucatorul mai are doar o carte in mana
- mai multe ferestre (start screen, player edit screen)
- carte de stopare a umflarii
- interfata grafica personalizata (FireAlpaca)

Pachetul nu contine jokeri.
